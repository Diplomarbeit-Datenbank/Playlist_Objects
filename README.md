# Playlist_Objects
![image](https://user-images.githubusercontent.com/87471423/130591591-39a14dac-1045-4049-a2b0-333c3335d691.png)

Eine Bibliothek für den Pixel Boy Musik Player, um eigene Playlists zu erstellen


# Aussehen: 
![image](https://user-images.githubusercontent.com/87471423/130592026-dba94583-d696-4f1b-a874-c857d95084f3.png)

# Funktionen: 
    o Anzeigen von der Playlist am Musik Player
    o Abspielen der Playlist am Musik Player
    o Löschen von bestehenden Playlists
    o Erstellen von Playlists
    o Bearbeiten von bestehenden Playlists
    o Hinzufügen von bis zu 999 Liedern pro Playlist
    o Automatische Playlist mit allen vorhandenen Liedern
    o Gif Animationen als Titelbild einer Playlist
    o Animierten Text, um den Namen der Playlist so lang machen zu können, wie man will


# Benötigte Librarys:
    o tkinter:  -> Wird benötigt für den filedialog (auswählen der gewünschten Dateien)
                -> sudo apt-get install python-tk (unter Windows vorinstalliert)
    o Ctkinter: -> Wird benötigt für die einzelnen Objekte, die am Playlist Canvas zu sehen sind
                -> Beschreibung unter folgendem Link:
    o Music_Widget: -> Wird benötigt um die Playlist zu starten und abzuspielen
                    -> Beschreibung unter folgendem Link:
    o shutil:       -> Für das verschrieben und kopieren von Dateien am Computer
                    -> pip install shutil
    o glob:         -> Um alle Ordnernamen eines Verzeichnisses herauszufinden
                    -> pip install glob
    o sys:          -> Für die Ausgabe von Fehlern in roter Farbe am Terminal
                    -> pip install sys
    o os:           -> Für das Erstellen von Ordnern in einem Verzeichnis
                    -> pip install os
